Présentation de l'app
- Permet de rechercher la bande d'annonce du film rechercher, avec le titre et la description du film
- Une liste apparait a droite de films que je pourrais aimer
- Par default l'app nous affiche les films les plus populaires
- Utilisation de l'API MovieDatabase

Pour lancer l'application
- ouvrir la console
- aller dans le dossier de l'app
- lancer la commande npm install
- lancer la commande npm start
